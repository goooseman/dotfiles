# lol

Plugin for adding catspeak aliases, because why not

## Enabling the plugin

1. Open your `.zshrc` file and add `lol` in the plugins section:

   ```zsh
   plugins=(
       # all your enabled plugins
       lol
   )
   ```

2. Restart your terminal session or restart the shell:

   ```console
   $ exec zsh
   $
   ```
